# Week 13 Lab 6: Quick Pick Lotto Application
A JavaFX application that generates secure random lottery numbers, runs concurrent draws using threads, stores results in a Hashtable, and saves them to a MySQL database.

## Features Implemented
- JavaFX GUI with input fields and buttons
- SecureRandom number generation (0-100)
- Hashtable for in-memory storage
- 5 concurrent threads for batch draws
- MySQL database integration
